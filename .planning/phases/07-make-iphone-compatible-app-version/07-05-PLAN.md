
    ```bash
    cd /home/rbrown/workspace/liminal && npx cap sync 2>&1 || echo "Cap sync skipped (ios/ not available)"
    ```

    **Step 4: Verify hover guards.**
    Grep for `hover:` Tailwind classes that lack `md:` guard. Per UI-SPEC constraint 1, all hover utilities should be wrapped with `md:hover:` or `@media (hover: hover)`. Run:
    ```bash
    grep -rn "hover:" frontend/components/ --include="*.tsx" | grep -v "md:hover" | grep -v "//.*hover" | head -20
    ```
    If any unguarded hover classes are found in components that render on mobile (BottomTabBar, SwipeableTaskCard, OfflineBanner, TaskCard), fix them by adding `md:` prefix.

    **Step 5: Verify input font sizes.**
    Per UI-SPEC constraint 2, all `<input>` and `<textarea>` must use `text-base` (16px) minimum.
    ```bash
    grep -rn "<input\|<textarea" frontend/components/ frontend/app/ --include="*.tsx" | grep -v "text-base\|text-lg\|text-xl" | head -20
    ```
    If any inputs use `text-sm` or smaller, update them to `text-base`.
  </action>

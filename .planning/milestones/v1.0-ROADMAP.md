**Status:** ✅ SHIPPED 2026-03-28

**Total Plans:** 22
**Git tag:** v1.0

## Overview

Transform Liminal from a traditional task manager into an ADHD-optimized productivity system by systematically removing friction, creating urgency, and providing dopamine rewards. The journey started by simplifying the existing interface (Phase 1), then layered on frictionless capture and immediate feedback (Phase 2), urgency visualization (Phase 3), gamification (Phase 4), working memory support (Phase 5), AI-powered prioritization (Phase 6), and a Capacitor iOS shell (Phase 7).

## Milestone Summary

**Key Accomplishments:**
1. Progressive disclosure task capture — title-only entry with auto-calculated scores eliminates capture friction
2. Global quick-capture FAB + Cmd/Ctrl+N + voice input with real-time transcription + confetti on completion
3. Full urgency system — deadline color gradients, live countdown timers, task aging, capacity summary, opt-in browser notifications
4. Gamification dashboard — animated StatsBar (daily count, streaks, personal best, impact pill), end-of-day wins toast
5. Forgiveness system — "where you left off" ring, interrupted task badge, auto-resume paused tasks on session restore
6. Inline AI "Do This Now" suggestion card with 15-min refresh, accept/dismiss, auto-switch to focus mode
7. Capacitor iOS shell — offline mutation queue (Dexie.js IndexedDB), bottom tab navigation, swipe-to-complete gestures

**Key Decisions:**
| Decision | Rationale | Outcome |
|----------|-----------|---------|
| Urgency over importance | ADHD brains respond to NOW/NOT NOW, not priority rankings | ✓ Core UX principle throughout |
| AI suggests, user overrides | Leverage LLM for smart defaults while preserving user agency | ✓ One-click dismiss/accept |
| Capture-first, refine-later | Reduce friction at moment of thought capture | ✓ Title-only + progressive disclosure |
| Gamification for momentum | Streaks and visual progress create sustained engagement | ✓ StatsBar + EoD toast |
| Warm orange for overdue (not red) | Red triggers RSD in ADHD users | ✓ Applied consistently |
| requestAnimationFrame for countdowns | Suspends cleanly, doesn't drift in background tabs | ✓ Better than setInterval |
| canvas-confetti for celebrations | GPU-accelerated, no custom CSS needed | ✓ Used throughout |
| Dexie.js for offline queue | TypeScript-native IndexedDB wrapper, fake-indexeddb for tests | ✓ Offline mutations work |
| Static export (output: 'export') | Required for Capacitor iOS — no server-side rendering | ✓ 89/89 tests pass |
